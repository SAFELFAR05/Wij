## Packages
framer-motion | For smooth animations (entry, hover effects, modal transitions)
clsx | Utility for conditional classes (often paired with tailwind-merge in shadcn)
tailwind-merge | Utility for merging tailwind classes

## Notes
The backend provides a single endpoint `/api/dramas` which returns a list of dramas.
Search functionality will be implemented client-side by filtering the fetched list.
Images will be served from the URLs provided in the API response.
